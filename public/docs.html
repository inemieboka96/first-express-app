Project Documentation

  • Express.js Setup: The server is created using express and listens on a configurable port.
  • Static Files: All files in the public folder are served automatically (HTML, CSS, images, etc.).
  • Routing: GET routes for /, /about, /contact, and /docs are handled by static files.
  • POST Routes: /submit and /data accept form and JSON data.
  • Middleware: Includes a logger and JSON body parser.
  • Error Handling: Custom 404 page for unknown routes.
